    JavaScript Chat

    The JavaScript Chat API supports chat from a browser on LiveEngage.

    The API is designed to help simplify chat on the browser with modern UI behavior. In addition to chat, this API also supports Pre-chat, Exit and Offline surveys.

    This API offers two main services:

    • Requesting or creating data via methods
    • Listening for data via events

    Methods for requesting or creating data

    Name Description
    Get Estimated Wait Time Returns the estimated wait time in seconds before a chat starts.
    Get Available Slots The capacity (or number of slots) in the chat system. This is the total number of chats that can be handled simultaneously by all online agents. Due to the fact that some agents can handle multiple chats, the number of slots is usually larger than the number of agents. Available Slots refers to the number of slots that are free at the current time.
    Get Availability Checks whether there are agents on this account that are available for chat. Returns an object with the property availability containing a true or false flag.
    Get Pre-chat Survey Retrieves the Pre-chat survey according to the criteria passed.
    Start Chat / Request Chat Queues a chat request to the agents. It has a number of possible properties that can be passed to specify the Agent, Skill or ServiceQueue requested and other information about the visitor.
    Add Line Sends a chat line to the agent. The response will echo the line sent.
    Set Visitor Typing Notifies the agent of the typing state of the visitor. The response echoes the request data.
    Set Visitor Name Updates the visitor’s name in the chat. The response echoes the original request data.
    End Chat Method to notify the agent that the visitor has ended the chat. The method publishes the new state as the response.
    Set Custom Variable Sets a custom variable for this chat session. This method can receive a single custom variable or a set of custom variables as properties of the customVariable object. The response echoes the variables set in the format that the server understands.
    Request Transcript Sends a copy of the chat transcript to an email address at the end of the chat session. The response will echo the email data.
    Get Exit Survey The Exit survey will be displayed after the chat has ended if one was configured for the account.
    Submit Exit Survey Submits the Exit survey results to the server. For more information about the Survey Object see Surveys.
    Get Offline Survey Retrieves the Offline survey to be displayed to the visitor, if no agent is available to chat.
    Submit Offline Survey Used to submit the Offline survey.
    Get State Returns the current chat state string according to the latest events received.
    Get Agent Login Name Returns the agent’s name string according to the latest events received.
    Get Visitor Name Returns the visitor’s name string according to the latest events received.
    Get Agent ID Returns the agent ID string according to the latest events received.
    Get Rt Session ID Returns the chat rtSessionId string according to the latest events received.
    Get Session Key Returns the chat session key string according to the latest events received.
    Get Agent Typing Returns the current agent typing boolean state according to the latest events received.
    Dispose Visitor Clears the chat session from memory, so on refresh it will not be automatically resumed.

    Events for listening for data

    Name Description
    On Load Fires when the chat API instance has been created and the DOM has loaded. It contains the current state of the API. If you are creating an instance of the chat API on page refresh or navigation, the state published here will be resume.
    On Init Triggers when you can request some information about the chat. The following methods are supported after this event has triggered: Get Available Slots, Get Estimated Wait Time, Get Availability, Get Pre-chat Survey, Get Offline Survey, Submit Offline Survey, Request Chat
    On Available Slots Triggered in response to Get Available Slots.
    On Estimated Wait Time Triggered in response to Get Estimated Wait Time.
    On PreChat Survey Triggered in response to Get PreChat Survey.
    On Start Triggered when a chat starts in response to Request Chat.
    On Stop Triggered when a chat has ended. Can be in response to End Chat or if the agent has ended the chat.
    On Request Chat Triggered in response to Request Chat.
    On Availability Triggered in response to Get Availability.
    On Transcript Triggered when a transcript has been requested from the server.
    On Line Triggered any time a line is added to the chat. The data here can be a single line or multiple lines.
    On State The current chat state.
    On Info Published when there is a change in the chat information.
    On Agent Typing Triggered when the agent typing status is changed. The Agent Typing property is a boolean value.
    On Exit Survey Triggered in response to the Get Exit Survey method.
    On Visitor Disposed Triggered in response to Dispose Visitor.
    On Offline Survey Triggered in response to the Get Offline Survey method.
    On Account To Account Transfer Triggered in response to the agent transferring the chat to another account.

    Technology

    The JavaScript Chat API is a REST style API.
